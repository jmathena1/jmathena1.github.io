R

  • the OG language. such a weird language to start with in hindsight but who cares. you can't always choose that. I still love R!
https://shiny.posit.co/r/getstarted/shiny-basics/lesson1/index.html
  • Shiny is R's framework for building apps. Great stepping stone to UI and web dev if you don't have that background. You can build Shiny apps entirely in R but also bring in modules that use HTML/CSS/JS as you want to and are able.
https://pkgs.rstudio.com/flexdashboard/index.html
  • Shiny like framework that is specifically for dashboards. We know how much analysts love building dashboards!
  • flexdashboard is great for small, regular reports. i used it for a couple monthly reports I needed that used to just be excel files. once i got the user satisfied with the visuals, i just changed the underlying data every month. pretty sure I used the `RMariaDB` library to connect to our AWS MySQL instance and just run the same query based on the month.
https://tidydatatutor.com/
  • let's you run R code in broswer. site will show you what your code is doing under the hood complete with visuals and diagrams. very good for learners that need a visual.
